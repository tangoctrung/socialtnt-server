##### SocialTNT 


Tạo file ```.env``` theo file ```.env.example``` rồi điền thông tin vào.
Sau đó chạy lệnh

```
    yarn install
    yarn start
```

Mở ```http://localhost:8800``` lên xem kết quả

Chú ý: port 8800 là mặc định nếu không xét port trong file ```.env```. Version node từ ```16.20.1``` trở lên.
